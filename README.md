**Mayvk – UniHack Project**

Mayvk is an AI-powered communication platform built during UniHack that enables real-time chat and voice interaction enhanced with sentiment analysis for job-seekers to prepare for interviews.

The project combines a modern frontend, backend APIs, and AI models to analyze user messages and provide emotional context in conversations.

**Features**

💬 Real-time chat using WebSockets

🧠 AI sentiment analysis for message understanding

🎙 Voice interaction support

⚡ Low-latency communication

🔌 Modular AI architecture

**Tech Stack**

Frontend: JavaScript / Web framework

Backend: Node.js / Python API

AI: HuggingFace Transformers (e.g., RoBERTa)

Communication: WebSockets
